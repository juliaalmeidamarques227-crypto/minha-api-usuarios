# API de Usuários

Essa é uma API REST simples que eu desenvolvi em C# usando ASP.NET Core para treinar os conceitos básicos de backend.

## O que ela faz

- Criar usuários  
- Listar usuários  
- Atualizar dados de um usuário  
- Deletar usuários  

## Tecnologias usadas

- C#  
- ASP.NET Core  

## Observação

Os dados são armazenados em memória, então ao reiniciar a aplicação tudo é resetado.

## Como rodar

```bash
dotnet run
